## 1. 🔴 Узлы Чебышёва / лемма о выборе узлов
**Лемма.** У полинома со старшим коэффициентом 1 максимум $|\cdot|$ на $[-1,1]$ не
меньше, чем у полинома Чебышёва (наименее уклоняющегося от нуля).
**Зачем:** в остаточном члене $\prod(x-x_i)$ — полином степени $n{+}1$ со старшим
коэф. 1; чтобы погрешность была минимальна, узлы = **корни полинома Чебышёва**.
$$\boxed{\,x_k=\cos\dfrac{\pi(1+2k)}{2m},\quad k=0,\dots,m-1\,}\quad(m=\deg=n{+}1)$$
- Линейная (2 узла, $m{=}2$): $\pm\frac{\sqrt2}{2}$.
- Квадратичная (3 узла, $m{=}3$): $\frac{\sqrt3}{2},\,0,\,-\frac{\sqrt3}{2}$.
- Отрезок $[a,b]$: $x=\frac{a+b}{2}+\frac{b-a}{2}\bar x$.

<a name="rk2"></a>
## 2. 🔴 Рунге–Кутта 2-го порядка: вывод погрешности (по углу / по координате)
Картинка: $l_1$ — касательная в $(x_0,y_0)$; по Эйлеру оценка в $x_1$ → $l_2$;
$l_3$ ≈ биссектриса; $l_4\parallel l_3$ через $(x_0,y_0)$.
$$y_1^{\text{угол}}=y_0+\tfrac h2\big[f(x_0,y_0)+f(x_0+h,\,y_0+hf_0)\big]\quad(\alpha_2=1)$$
$$y_1^{\text{коорд}}=y_0+h\,f\big(x_0+\tfrac h2,\,y_0+\tfrac h2 f_0\big)\quad(\alpha_2=\tfrac12)$$
**Вывод:** разложить $y(x_0+h)$ и $y_1$ в ряд Тейлора (функция 2 переменных),
вычесть. Первые члены ($h^0,h^1,h^2$) сокращаются ⇒ остаётся
$$\boxed{\,\varphi=O(h^3)\ \Rightarrow\ \text{метод 2-го порядка}\,}$$
Шпаргалки для вывода:
$y''=f'_x+f'_y f$; $f(x+\Delta x,y+\Delta y)=f+f'_x\Delta x+f'_y\Delta y+\dots$;
раскладывать **в окрестности $(x_0,y_0)$**, $\Delta x=h/2,\ \Delta y=\tfrac h2 f$ (коорд).

<a name="rk3err"></a>
## 3. 🔴 Метод 3-го порядка линейной комбинацией (для $y'=f(x)$)
$$R_{\text{угол}}=-\tfrac{h^3}{12}f''_{xx},\qquad R_{\text{коорд}}=+\tfrac{h^3}{24}f''_{xx}.$$
Главные члены отличаются только коэффициентом → убираем: $1\cdot(\text{угол})+2\cdot(\text{коорд})$:
$$\boxed{\,y_1=\tfrac13 y_1^{\text{угол}}+\tfrac23 y_1^{\text{коорд}},\qquad \varphi=O(h^4)\,}$$
Вывод: точное $3y(x_0{+}h)-y^{\text{угол}}-2y^{\text{коорд}}=O(h^4)$ (члены $h^3$ ушли).
**Вывод (общий $q{=}2$):** $\varphi$ через 3-ю производную, $\varphi(0)$ даёт главный
член $\frac{h^3}{3!}(1-\tfrac32\alpha_2)[\dots]$ ⇒ методы отличаются коэф. погрешности.

<a name="char"></a>
## 4. 🔴 Перенос: характеристики + граничные условия
$$u_t+c\,u_x=0\ \Rightarrow\ u=\varphi(x-ct),\quad \text{характеристики } x-ct=\text{const}.$$
Вдоль характеристики решение **постоянно** (переносится с условия). Наклон $1/c$.
**Где ставить условия** (область $x>0,t>0$): характеристика через угол $(0,0)$
делит область.
- **правее** разделяющей характеристики → решение с **начального** условия $u(x,0)$;
- **левее** → с **граничного** условия $u(0,t)$.
⇒ в ограниченной области нужны **и начальное, и граничное** условия.
**Согласованность:** в угловой точке начальное = граничное (непрерывность $u$).
*Пример:* $u_t+2u_x=0$, разделяет $x-2t=0$; $(1,10)$: $1{-}20<0$ → граничное;
$(10,1)$: $10{-}2>0$ → начальное.

<a name="rest"></a>
## 5. 🟠 Остаточный член интерполяции
$$\boxed{\,R_n(x)=f(x)-P_n(x)=\dfrac{f^{(n+1)}(\xi)}{(n+1)!}\prod_{i=0}^{n}(x-x_i)\,}$$
- Полином степени $n$ → остаток содержит полином степени **$n{+}1$**.
- $\xi$ существует, но неизвестна (теорема о среднем).
- **Тест программы:** полином степени $N$ имеет $N$-ю произв. $=N!$, $(N{+}1)$-ю $=0$.
  Подложить полиномы степеней $0,1,\dots,n$ → должны воспроизвестись точно;
  степень $n{+}1$ → погрешность известна точно.

<a name="cubic"></a>
## 6. 🟠 Кубический сплайн
Непрерывен с 1-й и 2-й производными; на отрезке — полином 3-й ст. $m_i$=1-я произв.,
$M_i$=2-я произв. в узле. Производная кубического = параболический сплайн.
Система для $M_i$ (через разделённые разности справа):
$$h_{i-1}M_{i-1}+2(h_{i-1}+h_i)M_i+h_iM_{i+1}=6\,f[x_{i-1},x_i,x_{i+1}],\ \ i=1..n{-}1$$
- **трёхдиагональная** ⇒ **метод прогонки**;
- **устойчивость** = диагональное преобладание ($2>1$);
- не хватает **2** условий (граничные на концах);
- $h=$const: $\ M_{i-1}+4M_i+M_{i+1}=\tfrac{6}{h^2}(f_{i+1}-2f_i+f_{i-1})$ (шаблон **1 4 1**).

<a name="rk4"></a>
## 7. 🟠 Рунге–Кутта: определение + классический РК4
$$k_i=hf\big(x+\alpha_i h,\ y+\textstyle\sum_{j<i}\beta_{ij}k_j\big),\quad y(x{+}h)\approx y+\textstyle\sum p_i k_i.$$
Константы $\alpha,\beta,p$ — из **минимума погрешности** ($\varphi(0)=\varphi'(0)=\dots=0$);
$s$=порядок, погрешность шага $O(h^{s+1})$, глобальная $O(h^s)$.
**Классический РК4** ($q=s=4$):
$$\boxed{\,y_1=y_0+\tfrac16(k_1+2k_2+2k_3+k_4)\,}$$
$$k_1=hf(x_0,y_0),\ k_2=hf(x_0{+}\tfrac h2,y_0{+}\tfrac12 k_1),$$
$$k_3=hf(x_0{+}\tfrac h2,y_0{+}\tfrac12 k_2),\ k_4=hf(x_0{+}h,y_0{+}k_3).$$
Порядок погрешности $O(h^4)$. Сам остаток выписывать не требуют.

<a name="euler"></a>
## 8. 🟠 Метод Эйлера + погрешность
$$\boxed{\,y_{i+1}=y_i+h\,f(x_i,y_i)\,}\quad(\text{касательная в }(x_i,y_i))$$
- Из ряда Тейлора: $y(x_0{+}h)=y_0+hy'+\tfrac{h^2}{2}y''(\xi)$.
- **Локальная** погрешность $\varphi=\tfrac{h^2}{2}y''=O(h^2)$ →
  **глобальная** $O(h)$ (1-й порядок: один порядок «съедается» накоплением).
- Структура всех методов: «значение в пред. точке + ещё что-то».

<a name="transp"></a>
## 9. 🟠 Перенос: определение + решение на бесконечной области
ЛДУ 1-го порядка. Замена $\xi=x-ct,\ \eta=x+ct$:
$\ \partial_t+c\partial_x\to 2c\,\partial_\eta$.
$$u_t+cu_x=0\ \Rightarrow\ u_\eta=0\ \Rightarrow\ \boxed{\,u=\varphi(x-ct)\,}$$
Бесконечная область ($t>0$): задать **начальное условие на всей оси** $u(x,0)=\varphi(x)$.
*Пример:* $u_t+u_x=0$, $u(x,0)=x^2{+}1$, $u(3,4)$: хар-ка $x{-}t={-}1$ → $u=(-1)^2{+}1=2$.

<a name="interp"></a>
## 10. 🟡 Интерполирование: Лагранж и Ньютон (Л15, Л16)
$P_n(x_i)=f_i$, по $n{+}1$ точке единственный (существование — конструктивно).
**Лагранж:** $P_n=\sum f_i\ell_i$, $\ell_i=\prod_{j\ne i}\frac{x-x_j}{x_i-x_j}$;
$\ell_i(x_k)=\delta_{ik}$, $\sum\ell_i\equiv1$, инвариантен к замене $x=a+bt$.
**Ньютон** (на разделённых разностях): $P_n=f[x_0]+f[x_0,x_1](x-x_0)+\dots$;
$f[x_0,x_1]=\frac{f_1-f_0}{x_1-x_0}$, далее рекуррентно; добавление узла = +1 слагаемое.
Лагранж и Ньютон — разные формы одного $P_n$ (обе для неравномерных сеток).

<a name="parab"></a>
## 11. 🟡 Параболический сплайн
На отрезке — полином 2-й ст., в узлах непрерывна **1-я** производная. Сначала
находят $m_i$ (1-е произв.) из системы $\frac{2}{h_i}(\dots)=m_i+m_{i+1}$, $i=0..n{-}1$;
не хватает **1** условия (на конце). Потом записывают сплайн на любом отрезке.

<a name="quad"></a>
## 12. 🟡 Эйлер: квадратурный способ, неявные методы
$y=y_0+\int_{x_0}^{x}f\,dx$, заменяем интеграл квадратурой:
- левые прямоуг. → явный Эйлер;
- правые прямоуг. → $y_{i+1}=y_i+hf(x_{i+1},y_{i+1})$ — **неявный**;
- трапеции → $y_{i+1}=y_i+\tfrac h2[f(x_i,y_i)+f(x_{i+1},y_{i+1})]$ — **неявный**.
«Неявный» = $y_{i+1}$ и слева, и справа.

<a name="high"></a>
## 13. 🟡 РК: высшие порядки, число вычислений правой части
$q{=}4$ — последнее, где $q=s$. При $s>4$ нужно $q\ge5$ этапов. Методы оценивают
по **числу вычислений правой части** $f$ (каждое $k_i$ = одно вычисление $f$).
РК4 — компромисс: хорошая точность при минимуме вычислений $f$.

<a name="nonhom"></a>
## 14. 🟡 Неоднородный перенос → ОДУ
$u_t+cu_x=f(x,t)$. Замена $\xi=x-ct,\eta=x+ct$ → $2c\,u_\eta=\tilde f(\xi,\eta)$ —
**обыкновенное ДУ** (интегрируется по $\eta$ при фикс. $\xi$). Вдоль характеристики
решение уже не постоянно, а находится интегрированием правой части.
