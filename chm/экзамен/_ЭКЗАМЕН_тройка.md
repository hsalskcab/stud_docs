## Блок 1. Интерполяция и сплайны (Л18–19)

**1. Что такое сплайн? Что такое кубический сплайн?**
Сплайн — кусочно-полиномиальная функция, на каждом отрезке сетки свой полином,
гладко «сшитый» с соседними. Кубический сплайн — на каждом отрезке полином
**3-й степени** с **непрерывными первой и второй производными** в узлах.
[Л19 0:13:05]

**2. Что такое параболический сплайн?**
На каждом отрезке полином 2-й степени с непрерывной первой производной. [Л18 1:12:39]

**3. Естественное (краевое) условие для сплайна.**
Когда **вторые производные на концах равны нулю** (S''(a)=S''(b)=0). При таких
условиях кубический сплайн даёт минимум интеграла от (f'')² — экстремальное
свойство. [Л19 0:41:26, 0:42:42]

**4. Задача сглаживания — что это?**
Исходные данные заданы **с погрешностями**; нужно построить **гладкую** функцию,
которая не проходит точно через точки, а приближает их. (В отличие от
интерполяции, где кривая проходит **через** узлы.) [Л19 1:04:37]

**5. Разделённая разность (1-го, 2-го порядка).**
Разделённая разность 1-го порядка: f(x₀,x₁) = (f(x₁)−f(x₀))/(x₁−x₀).
2-го порядка — разделённая разность от разделённых разностей:
f(x₀,x₁,x₂) = (f(x₁,x₂)−f(x₀,x₁))/(x₂−x₀). [Л18 2:07:18, 2:10:40]

## Блок 2. Численное дифференцирование (Л20)

**6. Что такое численное дифференцирование?**
Заданы значения функции **в узлах**; в этих узлах (или между ними) нужно найти
значения производной — первой, второй и т.д. [Л20 0:08:17]

**7. Что такое шаблон?**
Совокупность узлов, на которых ведётся рассмотрение, причём **один узел выделен**
— относительно него строится формула. [Л20 0:23:43]

**8. Остаточный член интерполяционного полинома.**
rₙ(x) = f(x) − Pₙ(x) — разность между точной функцией и интерполяционным
полиномом степени n. [Л20 0:09:28]

## Блок 3. Задача Коши, методы Эйлера (Л20–21)

**9. Что такое сетка?**
Система точек xᵢ, на которой ищется решение. Сеточная функция yᵢ — приближённое
решение в узлах. [Л21 1:09:26], [Л29 0:11:42]

**10. Метод Эйлера (формула).**
y_{i+1} = yᵢ + h·f(xᵢ, yᵢ). Геометрический смысл (по Эйлеру): на каждом шаге
идём по касательной к решению. [Л21 1:15:13]

**11. Главный член погрешности — что это?**
Старший (главный) по степени h член в разложении локальной погрешности метода;
определяет порядок точности. [Л21 0:33:09]

## Блок 4. Методы Рунге–Кутты (Л22–23)

**12. Определение метода Рунге–Кутты.** ⭐ (Спрашивают почти всегда — Л23 0:24:44)
Приближённое решение задачи Коши ищется как **линейная комбинация**:
y(x+h) ≈ y(x) + p₁k₁ + p₂k₂ + … + p_q k_q, где
k₁ = h·f(x, y), k_i = h·f(x + αᵢh, y + Σβᵢⱼkⱼ).
Константы αᵢ, βᵢⱼ, pᵢ определяются **из условия минимума погрешности** (чтобы
разложение φ(h) в ряд Тейлора начиналось с как можно более старшего члена).
[Л23 0:24:44–0:36:39]

**13. Порядок метода s.**
Число s, до которого совпадают разложения точного и приближённого решения;
определяется из условия минимума погрешности. [Л23 0:30:30, 0:32:04]

**14. Классический метод Рунге–Кутты.**
Метод 4-го порядка (q=4) — «классический», самый употребительный. [Л23 1:32:25]

**15. Контрольный член / правило Рунге (оценка погрешности).**
Сравнение решений с шагом h и h/2 для оценки погрешности и выбора шага
(«контрольный» член). [Л23 2:42:56], [Л22 оптимальный шаг]

## Блок 5. Краевые задачи: редукция, стрельба, сетки (Л24–27)

**16. Краевые условия 1, 2, 3 рода.**
1-го рода: задано значение функции (y(a)=A). 2-го рода: задана производная.
3-го рода: **линейная комбинация значения функции и производной**. [Л26 0:22:41]

**17. Метод редукции (вложенные методы).**
Сведение краевой задачи к нескольким задачам Коши; решение ищется в виде
y₁ = y₀ + p₁k₁ + … ; может решаться несколько линейных систем. [Л24 0:08:02, 0:12:52]

**18. Метод стрельбы — идея.**
Краевая задача сводится к задаче Коши с **пристрелкой** недостающего начального
условия так, чтобы попасть в краевое условие на другом конце. Эквивалентно
поиску минимума функционала / нахождению корня. [Л27 0:14:08, 2:02:36]

**19. Метод сеток (= метод конечных разностей).**
Производные заменяются разностными отношениями на сетке; получается система
алгебраических уравнений для y₀, y₁, …, yₙ. [Л25 1:26:14, 2:05:40]

## Блок 6. Вариационная постановка (Л26–27)

**20. Что такое функционал?**
Отображение, **аргументом которого служит функция** (а значением — число).
[Л26 1:13:09]

**21. Вариационная постановка краевой задачи.**
Решение краевой задачи **эквивалентно** поиску минимума соответствующего
функционала. [Л27 0:14:08]

**22. Самосопряжённая форма.**
Запись дифференциального оператора в виде (p·y')' + … — самосопряжённая форма.
[Л26 2:02:49]

## Блок 7. Теория разностных схем (ядро экзамена, Л29–33) ⭐⭐

> Л29 — «много определений, много новых понятий». Это самый важный блок на тройку.

**23. Сетка.** Некоторое множество **изолированных точек**. [Л29 0:11:42]

**24. Сеточная функция.** Функция, **определённая в узлах сетки**. (Дифференцировать
её нельзя — определена только в узлах.) [Л29 0:17:30]

**25. Норма в C (чебышёвская).** Максимум модуля сеточной функции по узлам.
Первая (сеточная) норма — связана с первой нормой вектора. [Л29 0:21:29], [Л30 1:28:10]

**26. Проектор.** Оператор, который **проектирует функцию на сетку** (берёт её
значения в узлах). [Л29 0:29:32]

**27. Разностный оператор L_H.** Оператор, действующий на сеточную функцию
(заменяет дифференциальный оператор разностным). [Л29 1:09:28]

**28. Разностная схема.** (Дословно Л29) **Совокупность сеточных уравнений,
заменяющих дифференциальное уравнение и краевые условия**, для определения
**приближённых сеточных значений** решения дифференциальной задачи. Если она
приводит к линейным уравнениям — **линейная** разностная схема. [Л29 1:04:40–1:07:18]

**29. Невязка (погрешность аппроксимации) δfₕ.** Подставляем в разностную схему
**проекцию точного решения** uₕ; то, что «отбрасывается» при замене
дифференциального оператора разностным, и есть невязка δfₕ.
⚠️ Не путать: **uₕ — проекция точного решения**, а **yₕ (uᴴ) — решение,
полученное из самой схемы**, это разные вещи. [Л29 1:21:00–1:23:42]

**30. Аппроксимация (на решении) и её порядок.** Схема аппроксимирует
дифференциальную задачу на решении u, если при подстановке проекции точного
решения **невязка δfₕ → 0 при h→0**. Аппроксимирует **с порядком σ**, если
‖δfₕ‖ ≤ M·h^σ, где M не зависит от h, σ>0. [Л29 1:32:05–1:36:19]

**31. Устойчивость.** (Дословно Л30) **Малое возмущение правой части вызывает
малое — равномерно по h — изменение решения**: для возмущённой правой части
fₕ+εₕ решение существует, единственно, и ‖zₕ − yₕ‖ ≤ C·‖εₕ‖, где C **не зависит
от h**; для **любой** правой части. [Л30 2:11:45–2:17:46]

**32. Сходимость и её скорость.** Схема сходится, если при h→0 норма разности
(проекция точного решения − сеточное решение yₕ) → 0. Скорость сходимости s,
если ‖uₕ − yₕ‖ ≤ C·hˢ (C не зависит от h, s>0). [Л29 1:28:24–1:30:35]

**32-bis. Теорема Филиппова (ключевая связка). ⭐**
**Из аппроксимации и устойчивости следует сходимость**, причём с тем же
порядком, что и порядок аппроксимации (константа C = K·M). [Л30 1:03:03–1:03:40]

**33. Принцип максимума.** Способ доказательства устойчивости **без матриц**:
оценка максимума решения через данные задачи. [Л30 1:32:14, 2:08:44]

**34. Матрица с диагональным преобладанием.** Модуль диагонального элемента
**больше суммы модулей внедиагональных** (в строке), часто с запасом δ. [Л30 1:05:16, 1:23:49]

## Блок 8. Эволюционные задачи: перенос, гармоники, параболика (Л28, 31–33)

**35. Явный шаблон.** На **новом (верхнем) слое только одна точка**. [Л31 0:20:12], [Л32 1:35:48]

**36. Неявный шаблон.** На новом слое **больше одной точки** (например три —
для аппроксимации второй производной по x). [Л32 2:08:12]

**37. Разности «вперёд / назад / центральная».** Назад — по двум точкам (текущая
и предыдущая); центральная — по **трём точкам** (симметрично). [Л28 2:58:59, 2:59:11]

**38. Уравнение переноса.** u_t + c·u_x = 0 (перенос со скоростью c). [Л31 0:25:01]

**39. Условие Куранта.** Условие устойчивости явной схемы для уравнения переноса
(связь шагов по t и x со скоростью c). [Л31 0:48:16]

**40. Абсолютная (безусловная) и условная устойчивость.**
Абсолютная/безусловная — схема устойчива при **любых** шагах. Условная — только
при выполнении ограничения на шаги (типа условия Куранта). [Л31 1:10:28, 1:11:23], [Л26 0:47:51, 1:05:45]

**41. Область влияния.** Множество значений, используемых для определения решения
в заданной точке. [Л30 1:37:38]

**42. Метод гармоник.** Исследование устойчивости подстановкой гармоники
(решение ищется в виде λⁿ·e^{ikx}); устойчивость — если |λ|≤1. [Л31 0:09:08, 1:12:17]

**43. Схема с весами.** Разностная схема, где новый и старый слои берутся с
весом σ (сигма — «вес»); ФИ — правая часть схемы (не путать с f). [Л32 2:48:13], [Л33 2:48:00]

## Что точно нужно уметь записать (формулы-минимум)
- Метод Эйлера: y_{i+1} = yᵢ + h·f(xᵢ,yᵢ)
- Общий вид метода Рунге–Кутты (k₁, k₂, линейная комбинация) — **наизусть**
- Краевые условия 1/2/3 рода
- Разностные производные: «вперёд», «назад», центральная; вторая производная
  (yᵢ₋₁ − 2yᵢ + yᵢ₊₁)/h²
- Невязка и порядок аппроксимации (O(hˢ))
